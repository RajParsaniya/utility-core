(1) Optional.ifPresent()

The Optional class provides an ifPresent() method that accepts a Consumer functional interface, which performs an action if a value is present, and does nothing if the value is absent.

Checked exceptions can be handled smoothly within the ifPresent() method without causing compile-time errors.
